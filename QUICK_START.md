# 快速开始 - 居民信息管理系统

## 🎉 项目已完成！

所有文件已经生成完毕，现在只需要推送到 GitHub 即可。

## 📋 推送步骤

### 第 1 步：打开 Git CMD

按 `Win + R`，输入 `cmd`，然后进入项目目录：

```bash
cd c:/Users/Administrator/WorkBuddy/20260312210322
```

### 第 2 步：初始化 Git 并提交

```bash
git init
git add .
git commit -m "Initial commit"
```

### 第 3 步：添加 GitHub 仓库

```bash
git remote add origin https://github.com/mmdjian/resident-management.git
```

### 第 4 步：推送到 GitHub

```bash
git push -u origin main
```

### 第 5 步：输入凭据

- **Username**: `mmdjian`
- **Password**: 输入你的 **Personal Access Token**

> 💡 如果没有 Token，访问：https://github.com/settings/tokens

## 🚀 推送成功后

1. 打开：https://github.com/mmdjian/resident-management
2. 点击 **Actions** 标签
3. 等待构建完成（5-10 分钟）
4. 构建成功后，下载 APK 文件

## ✅ 项目功能

- 📝 居民信息管理（添加、编辑、删除、查询）
- 📊 数据统计（按性别、年龄等）
- 📤 Excel 数据导出
- 🔐 生物识别登录
- 🎨 现代化 UI（Jetpack Compose）

## 📱 下载 APK

构建成功后：
1. 在 Actions 页面找到最新的构建任务
2. 点击构建任务
3. 滚动到底部，找到 **Artifacts**
4. 下载 `app-release` 文件
5. 解压后安装 APK
